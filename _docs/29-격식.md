## 격 식
- 홑처마 < 겹처마
- 민도리 < 굴도리
- 익공집 < 포집
- 포 : 주심포 < 다포
- 도리 : 주심 < 출목도리
- 기단 : 1(초벌대) < 2(두벌대) < 3(삼벌대) < 4 단
- 지붕 마루에 따라 격이 결정
   - 적새 3 < 5 < 7< 9 단
- 맞배집 < 합각집
- 민도리 < 소로 수장집
- 주초의 모양, 형식에 따라 격이 달라진다.
- 기둥의 목기
   - 쌍사
   - 외사
